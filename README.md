# Banking Customer Segmentation & Churn Risk Analysis
Power BI dashboard for RFM analysis and customer segmentation.

Project Overview
This project applies the "RFM (Recency, Frequency, Monetary)" model to analyze customer behavior.
The Power BI dashboard helps identify customer segments such as "Loyal, At Risk, Potential, and Churned", and highlights their revenue contribution.

Features
- KPI cards for total revenue, total customers, and average recency
- Pie chart showing "Customer Distribution by Segment"
- Bar chart showing "Revenue Contribution by Segment"
- Table with detailed "RFM scores (R, F, M, RFM_Score)"
- Interactive slicers for filtering by segment

Key Insights
- 25% Loyal customers contribute "60% of total revenue"
- 15% customers are "At Risk" and need re‑engagement
- Top 10 customers contribute "30% of revenue"

Tools & Technologies
- Power BI Desktop
- Dataset: RFM_results.csv
- Data modeling with RFM scoring logic
